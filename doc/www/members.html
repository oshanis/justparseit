Welcome to the Policy Parser!


This is the home page of the Policy Parser, the final project for 6.863 Spring 2008. The purpose of our project is to build a system for parsing law-enforcement policies, which are written in natural language, into a data model called Resource Description Framework (RDF). Once a policy is transformed into RDF, the user of our system can ask whether a particular sequence of events in a scenario adheres to or violates the policy.
